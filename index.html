진짜 네컷 비율로 만들고, 출력 모션까지

이번 버전은 흔히 쓰는 인생네컷 느낌의 출력 비율을 프리셋으로 넣고, 배경색 선택, 흑백 전환, 마지막 포토부스 출력 모션까지 붙인 버전이야.

1. 출력 크기 고르기

2×6 스트립, 4×6 카드, 모바일 미리보기 비율 중에서 고른다.

2. 캐릭터 PNG 준비

각 칸 크기에 맞춰 컷별 PNG를 만든다. 사람이 들어갈 부분은 투명하게 비운다.

3. 배경/색감 설정

배경색, 전체 배경 이미지, 흑백 전환을 선택한다.

4. 촬영 후 출력

사진을 찍거나 업로드한 뒤, 포토부스에서 뽑히는 듯한 모션 후 저장한다.
