# Real Four-Cut Booth

이번 버전은 실제 포토부스에서 많이 쓰는 네컷 출력 비율 느낌을 프리셋으로 넣은 버전입니다.

## 들어간 기능
- 2×6 스트립 기본 (600×1800)
- 2×6 스트립 고해상도 (1200×3600)
- 4×6 카드형 (1200×1800)
- 모바일 미리보기 (1080×1920)
- 컷별 캐릭터 PNG 업로드
- 배경색 여러 개 선택
- 전체 배경 이미지 업로드
- 흑백 전환
- 카메라 촬영 / 사진 업로드
- 포토부스 출력 모션
- PNG 저장

## 핵심 방식
위치 조정보다, 프리셋 크기에 맞는 캐릭터 PNG를 각 컷에 먼저 만드는 구조입니다.
