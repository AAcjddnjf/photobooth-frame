# 같이 찍는 프레임 웹사이트

서버 없이 작동하는 무료 정적 웹사이트입니다. 프레임 이미지와 사용자 사진은 브라우저 안에서만 처리되고, 어디로도 업로드되지 않습니다.

## 기능

- 프레임 이미지 업로드
- 사용자 사진 업로드
- 브라우저 카메라 촬영
- 사진 크기/위치 조정
- 1080×1920 PNG 저장
- 새로고침 또는 지우기 후 기록 삭제

## 프레임 이미지 준비

프레임은 1080×1920 PNG를 추천합니다. 사용자가 들어갈 네모칸 부분은 투명하게 비워두세요. 배경을 흰색으로 쓰고 싶으면 프레임 PNG 안에 흰 배경을 포함해도 됩니다.

## GitHub Pages에 올리는 법

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더 안의 `index.html`, `style.css`, `script.js`, `README.md`를 저장소에 업로드합니다.
3. 저장소의 Settings → Pages로 들어갑니다.
4. Branch를 `main`, 폴더를 `/root`로 선택하고 Save를 누릅니다.
5. 잠시 뒤 표시되는 Pages 주소로 접속합니다.

카메라 기능은 보통 HTTPS 주소에서만 정상 작동합니다. GitHub Pages 주소는 HTTPS라서 괜찮습니다.
