# Photo Frame Booth

브라우저에서만 작동하는 무료 포토부스형 웹앱입니다.

## 기능

- 시작 화면 → 프레임 크기 선택 → 프레임 PNG 업로드 → 사진 칸 위치 조절 → 촬영/업로드 → 결과 저장
- 카메라 모드: 1컷부터 4컷까지 카운트다운 후 순서대로 촬영
- 업로드 모드: 한 컷씩 선택하거나 4장을 한 번에 선택
- 최종 저장 크기: 1080x1920, 1080x1350, 1920x1080, 직접 입력
- 서버 없음. 사진은 브라우저 안에서만 처리됩니다.

## GitHub Pages

1. 이 폴더 안의 `index.html`, `style.css`, `script.js`, `README.md`를 저장소 루트에 업로드합니다.
2. Settings → Pages → Deploy from a branch → main / root → Save.
3. `https://아이디.github.io/저장소이름/` 주소로 접속합니다.

카메라는 반드시 `github.io` 주소에서 열어야 안정적으로 켜집니다.

## 프레임 만들기

프레임 이미지는 PNG 권장입니다. 사용자가 들어갈 사진 칸은 투명하게 비워 두세요. 프레임 속 인물, 꾸밈, 로고, 배경은 PNG 안에 넣으면 됩니다.
