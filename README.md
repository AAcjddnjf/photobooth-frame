# Sketch Booth

배포용처럼 보이도록 첫 화면에 가이드 캐릭터를 넣고 전체 디자인을 다시 다듬은 버전입니다.

## 들어간 것
- 첫 페이지 가이드 캐릭터
- 서비스형 랜딩 페이지
- 친절한 사용 흐름
- 사이즈 카드 선택
- 캐릭터 PNG 생성 프롬프트 자동 생성/복사
- 컷별 캐릭터 PNG 업로드
- 카메라 촬영 / 사진 업로드
- 배경색 / 배경 이미지 선택
- 출력 모션
- PNG 저장

## 배포
GitHub Pages에 index.html, style.css, script.js, guide-character.png, README.md를 올리면 됩니다.
