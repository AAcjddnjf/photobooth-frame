## 프레임 제작 규칙

프레임 PNG는 최종 크기와 같은 크기로 만드세요. 예: 1080×1920.
캐릭터와 배경은 불투명하게 두고, 사용자가 들어갈 사진 칸만 투명하게 비워야 합니다.
사이트에서 사진 칸 위치를 투명 영역에 맞추면, 촬영 중에도 프레임 안에서만 사람이 움직입니다.
